# Elto Pets

E-commerce website and landing page for premium pet products. Built with Next.js, Supabase, and Tailwind CSS using the Claude Code Webapp System.

## Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript (strict mode)
- **Styling**: Tailwind CSS + shadcn/ui
- **Database**: Supabase (PostgreSQL + Auth + Storage)
- **State**: React Query (server) + Zustand (client)
- **Animation**: Framer Motion
- **Deployment**: Vercel

## Getting Started

1. Clone: `git clone https://github.com/swetharavi2204-ops/elto-pets.git && cd elto-pets`
2. Start Claude Code: `cc`
3. Say: **"Generate a design system and fill in PROJECT.md"**
4. Then: **"Start Phase 01"**

## Phase Overview

| Phase | Sessions | What |
|-------|----------|------|
| 01 | 1-2 | Scaffold + shadcn/ui + Supabase schema |
| 02 | 3 | Auth middleware + pages |
| 03 | 10-20 | UI components (landing page + e-commerce) |
| 04 | 3 | Testing |
| 05 | 5-6 | API routes |
| 06 | 1-2 | Deploy |
